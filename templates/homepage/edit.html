Hello! First steps


To put the blog into operation, you need to execute the following commands:
  • $ python manage.py makemigrations
  • $ python manage.py migrate
  • $ python manage.py createsuperuser
  • Log in to the site with the username and password provided in the createsuperuser step
  • Customize the page as you wish or add a new post to the blog on the dashboard interface.
For more info how to setup a django app visit: Django documentation {% endblock %}
